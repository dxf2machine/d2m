1.Gestion des fichiers

                Le programme peut charger des fichiers DXF (AutoCAD ou autres applications qui peuvent créer des fichiers DXF.

1.1. Menu Fichier

Ce menu est accessible soit en haut à droite de l'écran:


Soit en effectuant un clique droit sur la racine de l'arbre en bas à droite:


1.1.1. Créer un nouveau dessin

Menu:

 

                Fichier  Nouveau

 

Touche de raccourci:

 

                Ctrl-N

 

Description:

 

                Cette commande permet de créer un nouveau canevas vierge. Le nouveau canevas est complètement vide (pas d'entités, un layer par défaut, pas de blocks).

1.1.2. Ouvrir un fichier

Menu:

 

                Fichier  Ouvrir

 

Touche de raccourci:

 

Ctrl-O

 

Description:

 

                Utilisez cette fonction afin d'ouvrir un fichier DXF. Ce fichier est chargé et est dessiné dans nouveau canevas.

 

1.1.3. Sauvegarder un dessin

Menu:

 

                Fichier  Sauvegarder sous

 

Touche de raccourci:

 

                Ctrl-S

Description:

 

                Cette commande permet de sauvegarder le dessin courant dans un fichier de type DXF. Utiliser le menu afin d'atteindre cette commande. Vous pouvez choisir de sauvegarder dans un nouveau fichier (dans ce cas spécifier le nom du nouveau fichier) ou de sauvegarder le dessin dans un fichier déjà existant (les données existantes dans ce fichier seront écrasées).

1.1.4. Renommer

Menu:

                Fichier  Renommer

 

Touche de raccourci:

 

                Ctrl-R

 

Description:

 

Cette commande permet de renommer le fichier du dessin courant. Une fenêtre de dialogue permet de saisir le nom de fichier désiré.

1.1.5. Exporter le dessin dans un fichier Image

Menu:

               

                Fichier  Exporter en image

 

Touche de raccourci:

               

                Ctrl-I

Description:

 

                Cette commande permet d'exporter le dessin dans un fichier de type fichier de type image.

Une fenêtre de dialogue vous permet de choisir l'emplacement, le type d'image (à choisir dans la liste déroulante) et le nom du fichier image à créer.


Attention: certain type ne sont pas géré, cela dépends du système d'exploitation utilisé et de la version de la plateforme JAVA utilisée.


 

 

 

 

 

 

1.1.6. Imprimer

Menu:

 

                Fichier  Imprimer

 

Touche de raccourci:

 

                Ctrl-P

 

Description:

 

                Cette commande permet d'imprimer le dessin courant. Une fenêtre de dialogue  vous permet de choisir l'imprimante et les options d'impression.

1.1.7. Fermer le dessin/programme

Menu:

 

                Fichier  Quitter

 

Touche de raccourci:

 

                Alt-F4

 

Description:

 

                Cette commande permet de fermer le programme et donc le dessin. Si le dessin n'a pas été sauvegardé toute modification non enregistrées au préalables seront perdues.

2. Options


2.1. le type de trait

On peut sélectionner le type de ligne (continue, pointillé etc.. ) en cliquant sur la liste déroulante appelée « current line type ». Pour utiliser cette option, aller dans l'onglet option puis cocher la case styles lignes (consomme de la mémoire).

2.2. l'épaisseur du trait

Dans le control panel on peut spécifier l'épaisseur du trait dans le champs current thickness (1 étant la valeur pour une épaisseur la plus fine disponible). Pour utiliser cette option, aller dans l'onglet option puis cocher la case styles lignes (consomme de la mémoire).

2.3. La couleur de l'objet


 On peut spécifier la couleur de l'objet à dessiner en cliquant sur la couleur désirée dans la palette de couleur.

By layer (on reprend celle du current) ou autre sélection

6.Outils complémentaires

6.1. Raccourcis clavier

 

Il existe plusieurs raccourcis clavier pouvant être utilisés conjointement avec la sélection dentités. Ci-dessous un récapitulatif de ces raccourcis et des  fonctions quils implémentent (dont le copier-coller).

 

6.1.1. Désélection

La touche échappe (Escape) désélectionne toutes les entités.

 

6.1.2. Tout Sélectionner

La touche A sélectionne toutes les entités.

 

6.1.3. Couper

La touche X coupe toutes les entités sélectionnées, et les place dans un conteneur temporel (Vecteur <myEntity> clipBoard).

 

6.1.4. Copier

La touche C copie toutes les entités sélectionnées, et les place dans un conteneur temporel (Vecteur <myEntity> clipBoard). Ainsi pour enlever une entité du clipBoard, il suffit de la désélectionner et appuyer de nouveau sur la touche C.

 

6.1.5. Coller

La touche P colle toutes les entités du conteneur temporel (clipBoard) dans le layer courant, à l endroit ou se trouve la souris. Il est ainsi possible de dessiner des lignes parallèles par exemple. Pour cela il suffit de sélectionner une ligne, appuyer sur la touche C pour la copier (ou X pour la couper), puis déplacer la souris, tout en appuyant sur P pour placer une copier de la ligne sur le dessin. On se retrouvera avec un ensemble de lignes parallèles espacées en fonction des décalages successifs de la souris.

 

6.1.6. Supprimer

La touche supprimer (Delete) supprime toutes les entités sélectionnées.

 

6.1.7. Block

La touche B déplace toutes les entités sélectionnées vers le block courant. [Note : Actuellement, la fonction ne marche que si le block courant est défini, on pourrait créer un nouveau block le cas échéant.]

 

6.1.8. Insert

 

La touche I permet de créer un block, y placer toutes les entités couramment sélectionnées, puis ajouter dans le layer courant un Insert appelant le Block précédemment crée. Les objets sélectionnés sont dorénavant contenus dans un block et référencés par un insert plutôt que directement dans un layer.

 

 

6.1.9. Dimension

La touche D permet de réaliser la même opération que ci-dessus, mais en plaçant les objets dans une entité dimension, cette fois.

Les touches I et D peuvent être utilisées successivement sur un même objet, ce qui permet de créer des inserts et dimensions sappelant les uns les autres (lutilité de cette fonction est discutable, mais permet surtout dimplémenter le principe des Block References décrit par la norme AutoCAD).
